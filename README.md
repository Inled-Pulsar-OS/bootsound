# pearos-boot-sound
A nice sound that greets you every time you boot your machine
## Installation steps (from source)
```sh
git clone https://github.com/Pear-Project/pearos-boot-sound
cd pearos-boot-sound
chmod +x install.sh
./install.sh
sync
shutdown -r now
```
