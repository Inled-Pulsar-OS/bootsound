# BootSound
Customizable boot and shutdown sounds for your Linux system.

## Features
- **GUI Installer**: Uses Zenity for an easy selection process.
- **Boot & Shutdown Sounds**: Configure sounds for both starting up and shutting down.
- **Predefined Sounds**: Includes classic sounds like Windows XP.
- **Custom Sounds**: Pick any audio file from your computer.

## Requirements
- `zenity` (for the GUI)
- `pipewire` / `pw-play` (for audio playback)

## Installation steps
```sh
git clone https://github.com/jaimegh-es/bootsound
cd bootsound
chmod +x install.sh
./install.sh
```
Follow the on-screen instructions to select your sounds.
